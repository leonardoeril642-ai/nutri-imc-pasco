# nutri-imc-pasco
NutriIMC es una app web gratuita para jóvenes de Pasco que calcula dietas y ejercicios según su IMC. Guarda el historial de peso de forma automática y propone planes económicos usando superalimentos nativos de la Selva Central (como sacha inchi, yuca y camu camu), impulsando la salud juvenil y la agricultura local desde Villa Rica.
